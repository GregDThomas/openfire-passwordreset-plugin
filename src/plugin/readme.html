Password Reset Plugin Readme

The Password Reset Plugin allows users to reset their own passwords if they have forgotten them. By supplying their Openfire username or associated email address, they will receive an email that will allow them to enter a new password.

Pre-requisites

  1. Supported databases are:
    • HSQLDB
    • PostgreSQL
    • MySQL
    However, adding additional database support is relatively simple, feel free to submit a PR!
  2. Openfire must not be configured with a read-only user authenticator - e.g. the plugin will work with the DefaultAuthProvider but not the LdapAuthProvider.
  3. Openfire must be correctly configured with suitable email settings. Login to the Openfire admin console and access Server -> Server Manager -> Email Settings.
  4. The user must have an email address configured.

Configuration

To configure the plugin, login to the Openfire admin console and access Server Settings -> Password Reset.
